# Species Profiles Reference

This directory contains detailed species profiles used by the core analysis service.

## Profile Structure

Each species profile includes:

### Basic Information
- Scientific name
- Common names
- Native range
- Growth characteristics

### Bonsai Suitability
- Difficulty level
- Recommended styles
- Size categories

### Care Requirements
- Watering needs
- Soil preferences
- Light requirements
- Temperature tolerance

### Pruning & Training
- Pruning timing
- Ramification techniques
- Wiring considerations
- Defoliation potential

### Repotting
- Frequency
- Root reduction percentage
- Soil mix recommendations

## Available Species Profiles

1. **Ficus retusa / Ficus microcarpa** (Chinese Banyan)
   - Tropical evergreen
   - Excellent for beginners
   - Styles: Root-over-rock, Informal upright, Forest

2. **Juniperus procumbens** (Japanese Garden Juniper)
   - Coniferous evergreen
   - Classic shohin material
   - Styles: Cascade, Semi-cascade, Informal upright

3. **Acer palmatum** (Japanese Maple)
   - Deciduous hardwood
   - Intermediate difficulty
   - Styles: Broom, Informal upright, Twin-trunk

4. **Ulmus parvifolia** (Chinese Elm)
   - Deciduous/semi-evergreen
   - Excellent for beginners
   - Styles: All styles suitable

5. **Pinus thunbergii** (Japanese Black Pine)
   - Coniferous evergreen
   - Advanced techniques
   - Styles: Formal upright, Informal upright, Slant

6. **Casuarina equisetifolia** (Australian Pine)
   - Tropical evergreen
   - Vietnamese specialty
   - Styles: Slant, Informal upright, Windswept

7. **Podocarpus macrophyllus** (Buddhist Pine)
   - Broadleaf evergreen
   - Intermediate difficulty
   - Styles: Formal upright, Informal upright, Forest
